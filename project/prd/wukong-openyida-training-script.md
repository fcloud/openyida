## PART 02 -- AI 时代的新范式（10分钟）

[切换到演示幻灯片]

### 人机协作的范式跃迁（3分钟）

刚才的演示，很多人可能会觉得——这不就是 AI 吗？ChatGPT 我也用过啊。

[停顿]

但我想告诉大家，刚才发生的事情，和大家平时用的 ChatGPT，有本质的区别。ChatGPT 只能跟你聊天，而刚才的 AI，是真的在帮你干活——它自己去创建应用、配置字段、发布上线，全程不需要你动手。

要理解这个区别，我们来看软件交互方式的四次跃迁。

第一个时代，DOS 命令行时代。黑底白字，你得记住命令才能用电脑。只有专业人士能操作。

第二个时代，GUI 图形界面时代。窗口、按钮、图标，你用鼠标点、用手指划。Windows、iPhone 把电脑和手机普及给了每个人。

第三个时代，LUI 语言界面时代。就是现在的 ChatGPT、钉钉悟空——你用自然语言跟 AI 对话。但这个阶段的 AI 主要还是在"聊天"。

第四个时代，CLI 工具时代——但这次不是给人用的命令行，而是给 AI 用的。AI 拿到了操作软件的"遥控器"，从聊天进化到了干活。

[停顿]

大家记住一个判断——未来每一个软件都会有两张"脸"。一张给人用，就是 GUI；一张给 AI 用，就是 CLI。这是行业共识——2026 年 3 月，一周之内，钉钉、飞书、企业微信三大平台同时开源了各自的 CLI。三天，三巨头同时出手。

### Programmable Enterprise（3分钟）

好，有了 CLI 这个遥控器，AI 能操作软件了。那下一个问题——企业里有多少东西是可以被 AI 操作的？

[停顿]

答案是：太多了。

审批流程、数据录入、报表生成、客户跟进、项目管理、会议纪要......企业日常运营中大量的工作，本质上都是重复性的、规则化的、可被编程的。

这就是 Programmable Enterprise 的概念——可编程的企业。以前这些工作需要人一个个去做，现在 AI 拿到了遥控器，可以批量自动化地完成。

[停顿]

而宜搭，恰好就是把企业中最常见的业务场景——表单、流程、报表、门户——全部变成了可编程的能力。

### Agent / Skill / CLI 三层架构（4分钟）

[展示 AI Agent 架构图]

大家看这张图。我用最白的话，一次性讲透三个关键概念。

[伸出三根手指]

第一个概念：Agent——数字员工。

Agent 是一个完整的"人"——有大脑能思考，有记忆能积累，有手脚能干活。钉钉的悟空，就是一个 Agent。

第二个概念：Skill——技能。

Agent 本身什么都能学，但它需要具体的技能才能干具体的活。就好比你招了一个很聪明的新人，他学习能力很强，但你得教他——怎么用公司的系统、怎么做报表。每教会他一样，他就多一项技能。OpenYida 就是一个 Skill——它教会 Agent "怎么操作宜搭低代码平台"。

第三个概念：CLI——操作工具。

就是我们刚才讲的"遥控器"。Skill 告诉 Agent"该做什么"，CLI 是 Agent"怎么操作"的具体工具。OpenYida 本身就是一套 CLI 命令——`openyida create-app` 创建应用、`openyida create-form` 创建表单、`openyida publish` 发布页面。

[停顿]

三者的关系一句话串起来——

Agent 是"人"，Skill 是"技能"，CLI 是"工具"。

悟空是那个聪明的员工，OpenYida 是教会他搭建应用的技能。

回到开场演示——"一句话做应用"的完整链路就是：

你说一句话 -> 悟空（Agent）理解你的需求 -> 调用 OpenYida（Skill） -> 通过 CLI 操作宜搭平台 -> 应用上线。

你只需要开口，剩下的事情，悟空带着 OpenYida 这个技能，帮你全部搞定。

## PART 03 -- 实战演示（25分钟）

### 环境准备与快速开始（5分钟）

[切换到演示页面]

好，在进入实战之前，我先用两分钟给大家讲清楚两个基础概念。

**什么是宜搭？**

宜搭，是钉钉官方的应用搭建平台。你不需要写代码，通过拖拖拽拽、选选配配，就能做出企业级应用。审批流程、数据报表、业务系统，都能做。它已经被全国超过一百万个组织在使用，安全性、稳定性经过了大规模验证。

**什么是 OpenYida？**

一句话——OpenYida 是 AI 和宜搭之间的桥。它是一个开源的 CLI 工具，让 AI 能直接操作宜搭平台。纯 Node.js 实现，无需 Python、无需 Playwright，安装即用。

[停顿]

安装方式非常简单，一行命令：

```
npm install -g openyida
```

装好之后，五步快速开始：

第一步，登录。`openyida login`。如果你用的是钉钉悟空环境，加一个参数：`openyida login --wukong`。

第二步，环境检测。`openyida env`。它会自动检测你用的是哪个 AI 编程工具。

第三步，初始化。`openyida copy`。把项目模板复制到你的工作目录。

第四步，对话创建。打开你的 AI 工具，用自然语言描述你要什么应用。

第五步，发布。`openyida publish`。一键发布上线。

[停顿]

OpenYida 本身是一个开源 Skill，不限制使用平台，支持主流 AI Coding 工具——悟空、OpenCode、VSCode、Qoder 等只是其中的例子，任何支持 CLI 的 AI 编程工具都能接入。

### 安全：四道防线（3分钟）

我知道在座的各位，看完演示，心里的第一反应可能不是"好厉害"，而是——"安全吗？"

这太正常了。作为管理者，安全永远排在第一位。

一共四道防线。

第一道，数据隔离。AI 只能在你授权的范围内操作，超出范围的事它碰不了。就像给员工划定职责范围——你负责 A、B、C，D 不归你管。

第二道，权限管控。不同角色、不同权限。老板看全部数据，部门经理看自己部门的，普通员工看自己的。AI 做出来的应用，权限管控和你手动做的完全一样。

第三道，审计追踪。每一步操作都有记录——谁做了什么、什么时间、改了哪些东西，全部可追溯。任何操作都可以撤回，不怕做错。

第四道，合规认证。宜搭是钉钉官方的企业级平台，安全合规能力经过全国大量企业验证。OpenYida 做出来的应用跑在宜搭上，享受同一套安全体系。

这不是草台班子，这是钉钉生态里的正规军。
