# 游戏大厅应用 PRD

## 应用信息

- **应用名称**：游戏大厅
- **应用类型**：宜搭自定义页面应用
- **页面名称**：游戏大厅

## 功能概述

类边锋游戏大厅风格的应用，包含斗地主、德州扑克、中国象棋、五子棋四款单机 AI 对战游戏，以及基于宜搭表单的全局排行榜。

## 页面列表

| 页面 | 类型 | 说明 |
|------|------|------|
| 游戏大厅 | 自定义页面 | 主页面，包含大厅 + 4 个游戏 + 排行榜 |
| 游戏成绩 | 表单页面 | 存储玩家游戏成绩，供排行榜查询 |

## 表单设计：游戏成绩

| 字段名称 | 字段类型 | 必填 | 选项/说明 |
|---------|---------|------|---------|
| 玩家昵称 | TextField | 是 | 最大长度 20 字符 |
| 游戏类型 | SelectField | 是 | 斗地主、德州扑克、中国象棋、五子棋 |
| 得分 | NumberField | 是 | 整数，最小值 0 |
| 胜负 | RadioField | 是 | 胜、负 |
| 对局时长 | NumberField | 是 | 单位：秒 |

## 自定义页面功能

### 大厅页

- Canvas 粒子星空背景动画
- 霓虹渐变标题"游戏大厅"
- 4 张游戏卡片（斗地主🃏、德州扑克♠️、中国象棋♟️、五子棋⚫）
- 卡片 hover 3D 倾斜 + 发光效果
- 排行榜面板（Tab 切换 4 个游戏，Top 10）

### 游戏页（通用）

- 顶部导航：返回大厅按钮 + 游戏名 + 当前得分
- AI 思考时顶部 loading 进度条
- 游戏结束弹窗：展示得分 + 填写昵称 + 提交排行榜

### 五子棋

- 15×15 棋盘，Canvas 绘制
- 人机对战，AI 使用 Minimax + Alpha-Beta 剪枝（深度 3）
- 落子波纹动画

### 中国象棋

- 9×10 棋盘，Canvas 绘制，楚河汉界
- 完整走法规则引擎
- 人机对战，AI 使用 Minimax（深度 2）+ 子力评估

### 斗地主

- 标准 54 张牌，3 人局（玩家 + 2 AI）
- 完整牌型识别（单/对/三带/顺子/飞机/炸弹/火箭）
- 出牌飞出动画

### 德州扑克

- No-Limit Texas Hold'em，玩家 vs 2 AI
- 完整流程：发牌→翻牌前→翻牌→转牌→河牌→摊牌
- 概率型 AI 决策

## 视觉风格

- 现代游戏大厅风：深色背景 + 霓虹渐变
- 主色：深空黑 `#0a0a1a`，金黄 `#ffd700`，青蓝 `#00d4ff`
- 卡片：毛玻璃效果 + 渐变发光边框
