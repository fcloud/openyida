# 施工管理数字化应用 - 设计文档

## 应用信息

| 配置项 | 值 |
| --- | --- |
| 应用名称 | 施工管理数字化 |
| 应用类型 | 宜搭混合应用（自定义页面 + 表单） |
| 核心目标 | 从"人管人"到"系统管人" |

## 功能架构

### 三大模块

| 模块 | 页面类型 | 核心功能 |
|------|---------|---------|
| 智慧工地 | 自定义页面 | 每日打卡（照片/视频）+ AI违规检测 |
| 节点验收 | 表单页面 | 开工/交底/隐蔽/竣工四方签字确认 |
| 工期预警 | 自定义页面 | 计划 vs 实际进度 + 延期自动推送 |

---

## 预警规则

### 触发条件
- 实际完成日期 > 计划完成日期

### 推送对象
- 分公司总经理（从项目信息表获取）
- 总部工程部（固定人员）

### 推送内容
- 项目名称
- 延期节点
- 延期天数
- 当前状态

---

## 页面设计

### 智慧工地（自定义页面）

- 工地打卡入口
- 实时打卡列表
- AI 违规检测展示（模拟）

### 节点验收（自定义页面）

- 项目选择
- 节点状态看板
- 待验收/已验收列表

### 工期预警（自定义页面）

- 延期项目列表
- 预警记录
- 状态更新

---

## 技术实现

- 使用 yida-create-app 创建应用
- 使用 yida-create-form-page 创建4个表单
- 使用 yida-create-page 创建3个自定义页面
- 使用 yida-custom-page 编写页面 JSX 代码
- 使用 yida-publish-page 发布页面
